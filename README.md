# Project_Bochechka
TG-бот книжный анализатор:
Анализ текстовых файлов, загружаемых пользователем (книги в формате .txt):
   1. Генерация облака слов из книги
   2. Построение графика появления персонажей в книге
   3. Определение авторства третьей книги на основе сравнения с двумя другими книгами известных авторов:
      3.1 График частотности длин слов для каждого автора
      3.2 График частотности длин предложений
      3.3 Сравнение лексикона с помощью критерия хи-квадрат
      3.4 Вычисление схожести набора используемых слов через коэффициент Жаккара
